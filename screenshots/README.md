# 演示截图说明

将以下截图放入本目录以便 README / 答辩使用：

| 文件名 | 内容 |
|--------|------|
| `01-analyze.png` | 分析页：URL 输入 + Agent 流程 + 日志 |
| `02-results.png` | 报告页：风险评分 + 图表 + 问题列表 |
| `03-live-mode.png` | 右上角 **Live 模式** 标识 |
| `04-demo-mode.png` | 右上角 **Demo 模式** 回退说明 |

## 如何截图

1. 运行 `python run_server.py`
2. 打开 http://127.0.0.1:8080/
3. 分析 `https://github.com/pallets/flask` 与 `本地示例仓库` 各截一张

可选：使用 Windows 截图工具或浏览器 DevTools 设备模式（1280×720）。
