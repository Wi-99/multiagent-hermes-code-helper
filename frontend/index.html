产品定位
面向非编程用户，粘贴 GitHub 链接即可获得结构化分析报告与示例修复 diff（可扩展框架演示）。

      
双模式
Live：后端 + GitHub API 读取公开仓库；Demo：sample_repos 或技术栈规则回退。

      
已知限制
不执行真实自动修复；Bandit/ESLint 等为可扩展占位；私有仓库需 Token（未内置）。

      
交付物
sample_output.md/json、demo/example-report.html、screenshots/README.md
